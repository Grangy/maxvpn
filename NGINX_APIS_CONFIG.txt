КОНФИГУРАЦИЯ NGINX ДЛЯ /apis/

Теперь структура такая:
- /api/ → порт 4000 (внешний API покупок)
- /apis/ → порт 3000 (Next.js API routes)

Добавьте в /etc/nginx/sites-enabled/grangy.ru.conf:

location /apis/ {
    proxy_pass http://127.0.0.1:3000;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
    proxy_set_header X-Forwarded-Host $host;
    proxy_cache_bypass $http_upgrade;
    
    # Таймауты
    proxy_connect_timeout 60s;
    proxy_send_timeout 60s;
    proxy_read_timeout 60s;
    
    # Буферизация
    proxy_buffering off;
}

ВАЖНО: Этот блок должен быть ПЕРЕД общим location /

После добавления:
sudo nginx -t
sudo systemctl reload nginx
